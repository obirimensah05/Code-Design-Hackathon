## 🔴 PROBLEM STATEMENT

**Professionals feel lost with AI tools.**

1. **Too many tools, no direction** — ChatGPT, Claude, Gemini, Midjourney, Cursor... which do I learn? Where start?
2. **Can't find role-specific learning** — Marketing needs different skills than developers
3. **No clear progression path** — What's beginner vs advanced?
4. **Knowledge is scattered** — Tips buried in YouTube, Twitter, Reddit. Disappears after a week
5. **Can't trust advice** — Is this person actually good at this?
6. **No community** — Lonely journey with no one to learn with

**Result:** Learners stay confused. Experts have nowhere to teach.

---

## 💡 SOLUTION

**PromptVault** = Peer-to-peer learning platform for **specific AI tools** at your skill level, in your industry.

### Core Value Proposition

**For Learners:**
✓ Find the exact AI tool you need RIGHT NOW
✓ Learn from people in your industry at your level
✓ Clear progression: Beginner → Expert
✓ Know who's actually good (real ratings, real stats)
✓ Community to learn with

**For Creators:**
✓ Share expertise and get recognized
✓ Build personal brand as AI expert
✓ Get paid (eventually) for teaching
✓ See real impact (views, ratings, who's learning)

## ✨ MICRO-INTERACTIONS & ANIMATIONS

### Philosophy
**Smooth and subtle. Like Framer or Figma. NOT cartoonish.**

**Animation speeds:**
- Fast: 150ms (hover effects, small movements)
- Medium: 250ms (page transitions)
- Slow: 400ms (dramatic moments)

### Specific Interactions

**1. Button Hover**
- Color deepens slightly (5-10% darker)
- Subtle shadow appears
- No scale change
- Duration: 150ms
- Easing: ease-out

**2. Video Card Hover**
- Border appears (1px, accent green)
- Slight shadow increase
- NO scale
- Duration: 150ms

**3. Leaderboard Rank Change (IMPORTANT)**
- When user moves up rank:
  - Previous card slides out (100ms)
  - New position card slides in (100ms)
  - Subtle green glow appears for 500ms
  - NO confetti (too much)
  
```
Animation:
- Slide out: transform translateY(20px), opacity 0 (100ms)
- Slide in: transform translateY(0), opacity 1 (100ms)
- Glow: box-shadow 0 0 20px rgba(0,255,136,0.3) (500ms)
```

**4. Points Earned (+10 pts)**
- Small "+10 pts" appears above video
- Floats up (20px)
- Fades out
- Color: accent green
- Duration: 1 second

**5. User's Leaderboard Entry**
- Very faint green background (5% opacity)
- NO animation (always visible, not flashy)
- Clear visual distinction

```
Style:
- Background: rgba(0,255,136,0.05)
- Border: 1px solid #E8E8E8
```

**6. Page Transition**
- Fade in: opacity 0 → 1 (200ms)
- Slight slide: translateY(10px) → translateY(0) (200ms)
- Simultaneous
- Easing: ease-out

**7. Form Input Focus**
- Border color changes to blue
- Subtle shadow appears
- Duration: 150ms

```
Focus state:
- Border: 2px solid #185FA5
- Shadow: 0 0 0 3px rgba(24,95,165,0.1)
```

**8. Success Toast**
- Appears from top-right
- Slides in: translateX(200px) → translateX(0) (150ms)
- Stays visible 3 seconds
- Slides out (150ms)
- Message: "+10 pts!" or "Video uploaded!"

**9. Real-time Update Badge**
- When leaderboard refreshes
- "Updated now" label appears
- Fades in (200ms), out after 3 seconds (300ms)
- Color: green text

**10. Loading State**
- Skeleton loader (gray boxes)
- Subtle shimmer animation
- Loop duration: 2 seconds

### Real-time Updates (IMPORTANT)

**When user completes a video:**

1. **Instant points (+10 pts)**
   - Toast appears: "+10 pts! 🎯"
   - Top-right corner
   - Auto-fades after 2 seconds
   - Leaderboard updates immediately (no refresh needed)

2. **Rank changes (if applicable)**
   - If rank improves:
     - Old card slides out (100ms)
     - New card slides in (100ms)
     - Subtle green glow (500ms)
     - Toast: "You moved to #2! 5 pts from #1"
   - If rank stays same: Numbers update, no animation
   - If rank drops: Numbers update, brief red tint

3. **Motivation messages**
   - "Only 5 pts to #1!"
   - "You're in top 10!"
   - "Keep going!" (subtle, not pushy)

### Weekly Reset
- Resets every Monday
- Message: "New week, new competition!"
- Last week's champion gets "Champion" badge
- Fresh leaderboard starts

### Psychological Design

**Why real-time works:**
- Excitement: "I moved up!"
- Motivation: "5 more pts and I'm #1"
- Social proof: Seeing others progress motivates you
- Friendly competition: Not toxic, just fun

### Mobile Leaderboard
- Simplified: Stacked vertically
- Your position at top (always visible)
- Top 3 below
- Scroll for more
- Same animations, optimized for touch
